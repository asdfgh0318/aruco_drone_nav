ArUco Drone Navigation

Vision-based indoor drone navigation using ceiling-mounted ArUco markers on Raspberry Pi Zero 2W

Overview

This system enables autonomous indoor drone flight using:

  • ArUco Markers - Ceiling-mounted visual fiducials for localization
  • OpenCV - Real-time marker detection and pose estimation
  • MAVLink - Communication with ArduCopter flight controllers
  • Raspberry Pi Zero 2W - Lightweight onboard computing

Quick Start

1. Start Camera Server on RPi

ssh pi@10.156.64.251
cd /home/pi/aruco_drone_nav
python3 tools/camera_server.py --port 8000

2. Launch Debug GUI on Local Machine

python3 tools/debug_gui.py --host 10.156.64.251 --port 8000

3. Calibrate Camera (if needed)

python3 tools/calibrate_remote.py --host 10.156.64.251 --port 8000

Key Features

Feature Description
Multi-Marker Navigation Navigate using multiple ceiling markers with world coordinate system
Mission Execution Execute waypoint missions from JSON files (VR-compatible format)
Position Estimation 6DOF pose estimation with marker-relative and world positions
Network Streaming MJPEG video streaming for remote debugging
Flight Recording Record trajectory and marker detections for analysis

Hardware Requirements

  • Companion Computer: Raspberry Pi Zero 2W
  • Camera: USB camera (640x480 @ 15+ FPS)
  • Flight Controller: ArduCopter-compatible (Pixhawk, etc.)
  • Markers: Printed ArUco markers (DICT_6X6_250, 20cm recommended)
Note: The system has been tested on RPi Zero 2W at 640x480 resolution achieving ~15 FPS detection rate.
